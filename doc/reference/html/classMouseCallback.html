Detailed Description

This file consists of template classes of mouse callback listeners.

Handle according to mouse behaviors leftClick and rightClick will be triggered only once in a time interval. When mouse button actually released, leftRelease or rightRelease will be triggered. move will be triggered when all above are not triggerd.
